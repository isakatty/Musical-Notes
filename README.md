# Lomuq v 1.0.0


## 프로젝트 소개
**`LO`g your practiced/lessoned `MU`si`Q`**

### 💫주요 기능
- **`달력`**
    - 달력에서 기록한 날짜에 맞춰  레슨 일지 확인
    - 레슨/연습 일지 작성 / 삭제 / 수정
- **`피드`** 
    - 기록한 음악 앨범 모아보기
    - 기록 수정 / 삭제 / 등록
- **`검색`** 
    - MusicKit을 활용한 음악 검색
